# Health-Sphere System

A digital healthcare platform for Lesotho hospitals and clinics.

## Features
- Digital patient records management
- AI-powered health predictions
- Secure information sharing
- M-Pesa mobile payments integration
- Appointment scheduling
- Ministry of Health reporting
